# TERAGRO Kurumsal Web Sitesi

Bu proje, TERAGRO markasının kurumsal dijital varlığını oluşturmak amacıyla geliştirilmiştir. Modern frontend teknolojileri kullanılarak hızlı, ölçeklenebilir ve SEO uyumlu bir yapı kurulmuştur.


## Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase (isteğe bağlı backend entegrasyonu)
- Vercel (deployment)



## Özellikler

- Responsive (mobil uyumlu) tasarım
- Kurumsal sayfa yapısı
- Hizmetler sayfası
- Referans / İş Ortakları bölümü
- İletişim sayfası
- SEO uyumlu meta yapı
- Optimize edilmiş production build
